# Plan: Chat Overview Page with Demo + Theme Switcher

## Goal

Build a chat overview page that mirrors the scheduler overview page pattern: a single `ChatBox` demo with a theme switcher (Default / Soft Edges / Neutral Vibes), using the same three themes as the scheduler.

---

## Architecture

The scheduler pattern works like this:

- `overview.md` embeds React components via `{{"component": "modules/components/overview/..."}}` syntax
- The main component (`MainDemo.tsx`) holds `useState` for the selected theme
- It calls `useTheme()` to read the outer docs-site light/dark mode, passes `mode` into theme factory functions
- A `ThemeProvider` wraps the demo area only (not the docs chrome)
- A MUI `<Select>` in a toolbar row lets users pick between named themes

For chat, we use the **same three themes** but adapt the `components` overrides to target chat-specific MUI component names (`MuiChatMessage`, `MuiChatBox`, etc.) instead of scheduler components.

### 4. `docs/src/modules/components/overview/chat/mainDemo/MainDemo.tsx`

The root orchestrator:

```tsx
import * as React from 'react';
import { createTheme, ThemeProvider, useTheme } from '@mui/material/styles';
import Paper from '@mui/material/Paper';
import Stack from '@mui/material/Stack';
import Select from '@mui/material/Select';
import MenuItem from '@mui/material/MenuItem';
import { ChatBox } from '@mui/x-chat';
import { createEchoAdapter } from '../../../../../../data/chat/material/examples/shared/demoUtils';
import {
  minimalConversation,
  minimalMessages,
} from '../../../../../../data/chat/material/examples/shared/demoData';
import { getSoftEdgesTheme } from '../theme/softEdgesTheme';
import { getNeutralVibesTheme } from '../theme/neutralVibesTheme';

type CustomThemeName = 'default' | 'softEdges' | 'neutralVibes';

const darkThemeManagement = {
  colorSchemes: { light: true, dark: true },
  cssVariables: { colorSchemeSelector: 'data-mui-color-scheme' },
};

const themeOptions = [
  { value: 'default', label: 'Default theme' },
  { value: 'softEdges', label: 'Soft edges' },
  { value: 'neutralVibes', label: 'Neutral vibes' },
];

export default function MainDemo() {
  const brandingTheme = useTheme();
  const [selectedTheme, setSelectedTheme] = React.useState<CustomThemeName>('default');
  const mode = brandingTheme.palette.mode;

  const getThemeByName = (name: CustomThemeName) => {
    switch (name) {
      case 'softEdges':
        return getSoftEdgesTheme(mode);
      case 'neutralVibes':
        return getNeutralVibesTheme(mode);
      default:
        return createTheme(darkThemeManagement, { palette: { mode } });
    }
  };

  return (
    <Stack spacing={1} sx={{ p: 1, width: '100%', mb: 6 }}>
      <Stack direction="row" justifyContent="flex-end">
        <Select
          value={selectedTheme}
          onChange={(e) => setSelectedTheme(e.target.value as CustomThemeName)}
          size="small"
          sx={{ minWidth: 140 }}
        >
          {themeOptions.map((opt) => (
            <MenuItem key={opt.value} value={opt.value}>
              {opt.label}
            </MenuItem>
          ))}
        </Select>
      </Stack>
      <ThemeProvider theme={getThemeByName(selectedTheme)}>
        <Paper variant="outlined" elevation={0} sx={{ height: 550, width: '100%' }}>
          <ChatBox
            adapter={adapter}
            initialActiveConversationId={minimalConversation.id}
            initialConversations={[minimalConversation]}
            initialMessages={minimalMessages}
            sx={{ height: '100%' }}
          />
        </Paper>
      </ThemeProvider>
    </Stack>
  );
}
```

> Note: `adapter` is created as a module-level constant (same pattern as `BasicAiChat.tsx`)

## Theme Adaptation Details

### softEdgesTheme for chat

- Same palette/shape/typography as scheduler's softEdgesTheme (borderRadius: 12, Inter font)
- Component overrides:
  - `MuiButton`, `MuiMenuItem`, `MuiList`, `MuiIconButton`: identical to scheduler version
  - `MuiChatMessage`: `bubble` slot gets `borderRadius: 12`; `roleUser` bubble uses primary color; `roleAssistant` bubble uses subtle grey
  - `MuiChatComposer`: `textArea` slot gets `borderRadius: 12`; `sendButton` gets same IconButton styling
  - `MuiChatConversationList`: `item` slot gets `borderRadius: 8`

### neutralVibesTheme for chat

- Same palette/shape/typography as scheduler's neutralVibesTheme (borderRadius: 8, General Sans font)
- Component overrides:
  - `MuiButton`: same bordered text variant
  - `MuiChatMessage`: `bubble` slot becomes white-bg bordered card (1px solid divider); `roleUser` gets a colored dot via `::before`
  - `MuiChatConversationList`: `item` slot gets bordered card appearance
  - `MuiChatComposer`: `root` gets border styling
