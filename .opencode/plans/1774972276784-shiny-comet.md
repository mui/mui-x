# Plan: Update Default ChatComposer Styling

## Goal

Update the default styles of `packages/x-chat/src/ChatComposer` components to match the target design in the reference image.

## Reference Image Analysis

The target design shows:

- **Container**: Rounded rectangle with a subtle border wrapping the _entire_ composer (all sides), no background fill (transparent/blends with page). No top-only border.
- **TextArea**: No visible border of its own — it blends seamlessly inside the container. No box-shadow on focus that conflicts with the outer border.
- **Toolbar**: Row below the textarea with two groups:
  - **Left**: action icon buttons (attach, emoji, GIF, mic) — muted gray icons, no background
  - **Right**: send button — dark gray/muted circle (NOT primary blue)
- **Toolbar layout**: `justifyContent: space-between` (not `flex-end`)
- **Send button**: Uses a muted/neutral dark color (e.g. `palette.action.active` or a dark gray) instead of `palette.primary.main`

## Files to Modify

### 1. `packages/x-chat/src/ChatComposer/ChatComposer.tsx`

Change the root styled `form` from top-border-only to a full rounded border:

```diff
- borderTop: '1px solid',
- borderTopColor: (theme.vars || theme).palette.divider,
- backgroundColor: (theme.vars || theme).palette.background.paper,
+ border: '1px solid',
+ borderColor: (theme.vars || theme).palette.divider,
+ borderRadius: theme.shape.borderRadius * 2,  // ~8px, more rounded
+ backgroundColor: 'transparent',
```

Also adjust padding slightly to accommodate the new rounded look (keep `padding: theme.spacing(1, 1.5)` or similar).

### 2. `packages/x-chat/src/ChatComposer/ChatComposerTextArea.tsx`

Remove the textarea's own border so it blends into the container:

```diff
- border: '1px solid',
- borderColor: (theme.vars || theme).palette.divider,
- borderRadius: theme.shape.borderRadius,
+ border: 'none',
+ borderRadius: 0,
```

Remove background (transparent — inherited from container):

```diff
- backgroundColor: (theme.vars || theme).palette.background.paper,
+ backgroundColor: 'transparent',
```

Remove focus border/shadow (the container provides the visual boundary):

```diff
- '&:focus': {
-   borderColor: (theme.vars || theme).palette.primary.main,
-   boxShadow: ...,
- },
+ // No focus border on textarea itself
```

Remove padding since the container already has padding (or reduce it):

```diff
- padding: theme.spacing(1, 1.5),
+ padding: theme.spacing(0.5, 0.5),
```

### 3. `packages/x-chat/src/ChatComposer/ChatComposerToolbar.tsx`

Change toolbar layout from right-aligned to space-between:

```diff
- justifyContent: 'flex-end',
+ justifyContent: 'space-between',
```

### 4. `packages/x-chat/src/ChatComposer/ChatComposerSendButton.tsx`

Change the send button color from primary to a neutral/muted dark gray:

```diff
- backgroundColor: (theme.vars || theme).palette.primary.main,
- color: (theme.vars || theme).palette.primary.contrastText,
+ backgroundColor: (theme.vars || theme).palette.action.active,
+ color: (theme.vars || theme).palette.background.paper,
```

And the hover:

```diff
- '&:hover': {
-   backgroundColor: (theme.vars || theme).palette.primary.dark,
- },
+ '&:hover': {
+   backgroundColor: (theme.vars || theme).palette.text.primary,
+ },
```

And the disabled:

```diff
  '&:disabled': {
    backgroundColor: (theme.vars || theme).palette.action.disabledBackground,
    color: (theme.vars || theme).palette.action.disabled,
```

## Follow-up: Padding + Muted Background (addendum)

**File:** `packages/x-chat/src/ChatComposer/ChatComposer.tsx`

Two changes to `ChatComposerStyled`:

1. **Padding**: Increase from `spacing(1, 1.5)` → `spacing(1.5, 2)` for more breathing room
2. **Background**: Change from `'transparent'` → `(theme.vars || theme).palette.action.hover` — this is MUI's standard muted/subtle overlay color (a very light gray on light themes, a slightly lighter surface on dark themes)

## Summary of Style Changes

| Component                | Current                            | Target                                    |
| ------------------------ | ---------------------------------- | ----------------------------------------- |
| `ChatComposer` (root)    | Top border only, paper bg          | Full rounded border, transparent bg       |
| `ChatComposerTextArea`   | Bordered box, paper bg, focus ring | Borderless, transparent bg, no focus ring |
| `ChatComposerToolbar`    | `justify-content: flex-end`        | `justify-content: space-between`          |
| `ChatComposerSendButton` | Primary color circle               | Muted gray circle                         |

## Verification

After making changes:

1. Run TypeScript check: `pnpm --filter "@mui/x-chat" run typescript`
2. Run unit tests: `pnpm test:unit --project "x-chat" --run`
3. Visual check: Start the docs dev server and view the basic chat demo at the composer page
4. Run `pnpm proptypes` and `pnpm docs:api` if any prop types changed (they shouldn't for style-only changes)
