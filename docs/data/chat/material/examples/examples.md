# Material examples

<p class="description">End-to-end examples showing production-ready patterns with <code>@mui/x-chat</code> styled components.</p>

These examples combine multiple Material layer features into complete, self-contained demos.
Each one answers a real product question and can be used as a starting point for your own implementation.

## Start here

- [AI Assistant](/x/react-chat/material/examples/ai-assistant/) — production AI copilot with streaming, markdown, reasoning, and tool calls
- [Customer support inbox](/x/react-chat/material/examples/customer-support-inbox/) — two-pane layout with conversations, unread badges, and typing indicators
- [Thread-only copilot](/x/react-chat/material/examples/thread-only-copilot/) — minimal embedded assistant without a conversations sidebar

## Theming and customization

- [Dark mode chat](/x/react-chat/material/examples/dark-mode-chat/) — dark theme with palette.Chat token overrides
- [Branded chat](/x/react-chat/material/examples/branded-chat/) — custom colors, border-radius, and typography for a branded look
- [RTL chat](/x/react-chat/material/examples/rtl-chat/) — right-to-left layout with Arabic text

## Advanced patterns

- [Skeleton loading](/x/react-chat/material/examples/skeleton-loading/) — all loading and bootstrapping states
- [Custom message actions](/x/react-chat/material/examples/custom-message-actions/) — custom action buttons on messages

## Recommended progression

1. Start with [AI Assistant](/x/react-chat/material/examples/ai-assistant/) to see the full Material surface in action.
2. Move to [Customer support inbox](/x/react-chat/material/examples/customer-support-inbox/) for the most common multi-conversation layout.
3. Choose a specialization:
   - [Dark mode](/x/react-chat/material/examples/dark-mode-chat/) or [Branded](/x/react-chat/material/examples/branded-chat/) for theming
   - [Skeleton loading](/x/react-chat/material/examples/skeleton-loading/) for loading states
   - [Custom message actions](/x/react-chat/material/examples/custom-message-actions/) for interaction patterns
