# Chat - Unread Marker

<p class="description">A divider that separates already-read messages from those that arrived while the user was away.</p>

{{"component": "@mui/internal-core-docs/ComponentLinkHeader"}}

## Overview

`ChatUnreadMarker` renders inline inside the message list when the active conversation has unread messages. The marker is positioned at the first unread message and disappears once the user catches up.
