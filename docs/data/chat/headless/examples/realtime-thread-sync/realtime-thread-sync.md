# Realtime thread sync

<p class="description">Use realtime events to add, update, remove, and rename conversations and messages.</p>

This recipe focuses on collection synchronization:

- `message-added`, `message-updated`, and `message-removed`
- `conversation-added`, `conversation-updated`, and `conversation-removed`
- active conversation resets when a thread disappears
