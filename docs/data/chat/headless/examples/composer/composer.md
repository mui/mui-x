# Composer

<p class="description">Build a custom draft area with attachments, preview URLs, and IME-safe submit behavior.</p>

This recipe uses `useChatComposer()` directly with plain DOM controls.
It demonstrates:

- draft value updates
- attachment add and remove
- image preview URLs
- `clear()` and `submit()`
- IME composition tracking through the headless store
