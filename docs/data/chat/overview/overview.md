## Accessible by default

Keyboard navigation, labeled landmarks, and screen reader announcements work out of the box: the message list is a single Tab stop with roving focus, and streaming responses are announced once at start and completion—never per token. Read the full model on the [Accessibility](/x/react-chat/accessibility/) page.
